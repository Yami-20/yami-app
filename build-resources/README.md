# Build Resources

Place your app icons here before building:

- icon.png  — 512×512 PNG, used for Linux and Android
- icon.ico  — Windows .ico file (multi-size, 16/32/48/256px)

## Quick icon generation

If you have a 512×512 PNG called icon.png, you can generate the .ico with:

  npm install -g png2icons
  png2icons icon.png icon

Or use https://convertico.com to convert online.

Yami's brand colors: #a78bfa (purple) → #f472b6 (pink)
